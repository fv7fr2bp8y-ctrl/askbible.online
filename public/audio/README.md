# Аудио файлове

Сложи тук аудио записите на стиховете (`.mp3`, `.m4a`, `.ogg`).

Препоръки:
- Формат: **MP3** (най-съвместим) или **m4a/AAC**.
- Качество: 128–192 kbps е достатъчно за говор.
- Име на файла: без интервали и кирилица, напр. `dom.mp3`, `liubov.mp3`.

После добави стиха в `src/data/poems.ts` с път `/audio/dom.mp3`
(пътят започва с `/` и НЕ съдържа `public`).
